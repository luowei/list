版本控制系统git的使用

包括：
私有公共repository整合到github公共的repository；
intellij中代码使用git提交推送代码与更新拉取代码；

步骤：
a.首先在本地添加一个‘github’的branch,并切换到该分支下,命令:git branch -b github

b.在github上给项目再添加一个远程分支,使用命令:git remote add github https://github.com/luowei/mvn-test.git

c.再把github远程仓库信息更新到本地:gitpull github master;
d.最后使用命令git push github github:master,将本地的修改推送到github远程仓库的master分支上;

e.测试:
使用命令git push github github:github,将本地的’github’分支推送到github远程仓库的github分支上。 使用命令git push github :github,将本地的一个空的分支推送到github远程仓库的github分支上,这样就删除github远程仓库的github分支。
