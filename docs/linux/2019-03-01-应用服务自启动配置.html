通过crontab 把 mock-server加入自启动
执行命令：$ crontab -e 并添加

# 以luowei用户执行

# 重启后启动mock-server
@reboot sh ( mock-server --dir=/var/www/mock.wodedata.com --port=8888 --address=0.0.0.0 & )

# 每月1日零晨3点清理mock-server日志
0 3 1 * * rm /var/www/mock.wodedata.com/*.log

mongodb自启动
执行命令：$ crontab -e 并添加 @reboot sudo service mongod start
